На найденном сундуке наши Роботы обнаружили хитрую панель в виде решетки с найденном точками. Решетка состоит из множества точек, расположеных в форме квадратов и некоторые линии соединяют соседние точки. Для того чтобы открыть сундук надо найти код, который совпадает с числом квадратов, которые образуют линии на решетке. Для примера на картинке вы можете видеть три квадрата -- два маленьких и один средний.



Точки пронумерованы числами от 1 до 16. Каждая линия представлена, как массив двух чисел, обозначающих соответствующие точки.

Входные данные: Массив линий, как список (list) списков (list). Каждый список состоит из двух целых чисел от 1 до 16.

Выходные данные: Количество квадратов, сформированных данными линиями, как целое число (int).

Примеры: 

checkio([[1, 2], [3, 4], [1, 5], [2, 6], [4, 8], [5, 6], [6, 7],
     [7, 8], [6, 10], [7, 11], [8, 12], [10, 11],
     [10, 14], [12, 16], [14, 15], [15, 16]]) == 3
checkio([[1, 2], [2, 3], [3, 4], [1, 5], [4, 8],
     [6, 7], [5, 9], [6, 10], [7, 11], [8, 12],
     [9, 13], [10, 11], [12, 16], [13, 14], [14, 15], [15, 16]]) == 2

Связь с реальной жизнью: Эта простая головоломка использует основы распознавания образов. Для более сложных случаев, вы можете усовершенствовать вашу программу и использовать ее для распознания более сложных фигур.
